조리법[편집]

🍳한국의 계란 요리 대한민국 국기

  1. 프라이팬을 살짝 달군 뒤 식용유(혹은 버터)를 두른다.[5]
  2. 프라이팬의 온도가 적당히 올라가면 계란을 깨서 넣는다.
  3. 타지 않도록 최대한 주의한다.
  4. 취향에 맞추어 뒤집으면서[6] 굽는다.
  5. 취향에 따라 소금 등의 조미료[7]나 향신료[8]를 넣는다.
