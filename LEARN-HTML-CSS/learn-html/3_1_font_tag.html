🍔 / Hamburger

패티를 구운 후 다양한 부재료와 함께 빵 사이에 끼워 먹는 음식이다.
전세계에서 가장 유명한 미국 요리이자 세계화의 바이블급 인지도를 자랑하는 넘버원 패스트푸드로,
세계로 퍼지는 과정에서 각종 문화가 뒤섞이다보니 이것을 바탕으로 만든 온갖 바리에이션 음식들이 등장했다.
때문에 어원과는 무관하게 햄버거처럼 만든 겹빵 음식이라는 의미인 버거(Burger)라는 접미형 신조어도 만들어냈을 정도.
이러한 접미방식이 처음으로 사용된 음식은 치즈버거라고 한다.
